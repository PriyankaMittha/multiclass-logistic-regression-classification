# Multiclass Logistic Regression Classification

This project demonstrates how to solve a **multiclass classification problem using Logistic Regression** in Machine Learning. The model is trained and evaluated using Scikit-Learn.

---

## 📌 Project Overview

Logistic Regression is a supervised machine learning algorithm used for **classification tasks**. While basic Logistic Regression is used for binary classification, it can also be extended to solve **multiclass classification problems**.

In this project, we implement **Multiclass Logistic Regression** and evaluate the model using different performance metrics.

---

## 🧠 What is Multiclass Classification?

Multiclass classification is a type of classification problem where the target variable has **more than two classes**.

Example:

- Class 0
- Class 1
- Class 2

Logistic Regression can handle this using strategies like:

- **One-vs-Rest (OvR)**
- **Multinomial Logistic Regression**
